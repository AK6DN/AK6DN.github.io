AK6DN Home Grown Diagnostic Files
a k 6 d n

This directory contains listings and executables of AK6DN's home grown PDP-11 diagnostics.

A single TU58 (or RX02) bootable XXDP image is available that contains all the .BIN/.BIC file images. It is 11XX_9.DSK (or .RX2) on the TU58 (RX02) disk images page.

For each diagnostic (eg, MEMX.*) there is a plain text listing (MEMX.LST) of the assembled diagnostic, and an executable binary image (either MEMX.BIN or MEMX.BIC, as appropriate).

Most (but not all) tests can be run under SIMH. When possible, a SIMH control (eg MEMX34.ini) is supplied.

Recently added is a .tar.gz file of my entire local diagnostics directory with all source and Makefiles included for those folks who want to see under the hood or make changes to any source files for their own specific needs.
