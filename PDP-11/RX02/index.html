RX02 XXDP Image Files
a k 6 d n

This directory contains bootable XXDPv25 RX02 disk image files of PDP-11 diagnostics.

For each image file (eg, 1134_1.RX2) there is a plain text listing (1134_1.TXT) of the directory of the image.

Each of the .RX2 image files are exactly 512,512B (1001 blocks of 512 bytes) of data and are bootable on a suitably equipped PDP-11 and a physical RX02 drive and/or drive emulator package (such as the Arduino RX02 drive emulator shield described here: RX02_EMULATOR.

The files are exact sector for sector images of a physical RX02 floppy disk (track0/sector0 thru track76/sector26) and could be imaged to a physical floppy using the appropriate software (an exercise for the reader...).

The images files can also be mounted within PDP-11 SIMH as an RX02 image file (using device driver RY) and booted.

Lastly, the whole point of this exercise is that the image files can be copied to a MicroSD card as is, and used with the above mentioned Arduino RX02 emulator to boot and run diagnostics on a PDP-11 system.

For the details on running each diagnostic see PDP-11 Diagnostic Handbook as the definitive reference.
